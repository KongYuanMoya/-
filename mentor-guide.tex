\section{平台简介}

\begin{goldbox}
\textbf{耀考教育} 是专注于国际课程辅导的在线教育平台，连接海内外名校导师与有志于 IB、AP、A-Level、IGCSE 课程的学生。平台严格筛选来自牛津、剑桥、LSE、港大等知名高校的优质导师，采用 \textbf{ClassIn} 专业在线教室提供高清 1 对 1 互动辅导，课后支持免费回看，确保每位学生获得高质量学术支持。
\end{goldbox}

\subsection{我们的使命}

帮助有学科专长的优秀教育者，以专业、灵活的方式为学生提供高价值辅导服务，同时实现自身的知识变现与职业成长。

\subsection{平台核心特色}

\begin{itemize}
  \item \textbf{严选导师}：全部导师经过严格背景审核，来自牛津、剑桥、LSE、港大等国内外知名高校
  \item \textbf{灵活预约}：7$\times$24 小时在线预约，自由选择上课时间
  \item \textbf{专业教室}：采用 ClassIn 在线教室，高清视频 + 互动白板 + 屏幕共享，课后可免费回看
  \item \textbf{品质保障}：首次试听不满意全额退款，无忧学习保障
\end{itemize}

\section{申请条件}

\subsection{符合条件的申请者}

\begin{infobox}
平台严选每一位入驻导师以保障学生获得高质量辅导。如果你是海内外名校的在读学生或毕业生，且具有扎实的学科知识和良好的教学能力，欢迎申请。
\end{infobox}

以下条件符合越多，审核通过概率越高：

\vspace{4pt}
\begin{tcolorbox}[
  enhanced, colback=GreenLt, colframe=GreenOk,
  boxrule=1pt, arc=6pt,
  left=12pt, right=12pt, top=10pt, bottom=10pt
]
\begin{itemize}[leftmargin=1.2em, itemsep=5pt]
  \item[\checkyes]\ \textbf{学历要求}：本科及以上学历，名校背景（国内 985/211、海外 QS Top 100）优先
  \item[\checkyes]\ \textbf{课程熟悉度}：熟悉至少一个课程体系（IB / AP / ALevel / IGCSE）
  \item[\checkyes]\ \textbf{辅导经验}：有过家教、助教或相关辅导经历；或为相关专业在读/应届毕业生
  \item[\checkyes]\ \textbf{授课设备}：具备稳定网络（建议带宽 $\geq$ 20Mbps）、电脑及摄像头
  \item[\checkyes]\ \textbf{时间投入}：每周可投入至少 \textbf{10 小时}以上的稳定授课时间
  \item[\checkyes]\ \textbf{语言能力}：中文授课或英文授课均可；双语导师享有更高曝光优先权
\end{itemize}
\end{tcolorbox}

\begin{infobox}
\textbf{注意：}综合评估学历背景、专业能力与辅导经历，符合条件即可申请。
\end{infobox}

\subsection{不符合条件的情形}

\begin{tcolorbox}[
  enhanced, colback=RedLt, colframe=RedErr,
  boxrule=1pt, arc=6pt,
  left=12pt, right=12pt, top=10pt, bottom=10pt
]
\begin{itemize}[leftmargin=1.2em, itemsep=5pt]
  \item[\checkno]\ 无法提供\textbf{有效身份证明}文件（身份证、护照或居留证）
  \item[\checkno]\ 完全没有目标课程体系的学习或辅导经验
  \item[\checkno]\ 网络条件不稳定，无法保证视频课质量
  \item[\checkno]\ 最终学历为高中及以下（特殊技能类科目须另行沟通）
  \item[\checkno]\ 提交虚假信息或伪造材料
\end{itemize}
\end{tcolorbox}

\begin{warnbox}
\textbf{重要警告：}一旦发现虚假信息，账号将被\textbf{永久封禁}，情节严重者平台保留追究法律责任的权利。
\end{warnbox}

\subsubsection{步骤一：注册账号}

访问耀考教育官网（\href{https://www.yaokaojiaoyu.com}{www.yaokaojiaoyu.com}），选择任一方式注册：
\begin{itemize}
  \item \textbf{手机号注册}：手机号 \arrowright\ 短信验证码 \arrowright\ 设置密码
  \item \textbf{邮箱注册}：邮箱地址 \arrowright\ 邮件验证码 \arrowright\ 完成验证
\end{itemize}

在平台注册后，填写真实姓名、教育背景及辅导科目。注册后即可提交导师申请，审核通过后即可开课。

\subsubsection{步骤二：上传认证材料}

详细材料清单请参阅第~\ref{sec:materials}~节。核心要求：
\begin{itemize}
  \item \textbf{必须提交}：身份证明（护照或身份证正反面）
  \item \textbf{强烈推荐}：本科/硕士/博士毕业证或在读证明
  \item 文件清晰，文字可辨，四角完整；格式：JPG / PNG / PDF
\end{itemize}

\subsubsection{步骤三：提交审核（1 次点击）}

确认材料无误后，点击「提交审核」。提交后：
\begin{itemize}
  \item 在「个人中心 \arrowright\ 认证状态」可实时追踪审核进度
  \item 平台将在 \textbf{3 个工作日内}完成审核
\end{itemize}

\subsubsection{步骤四：审核通过}

通过后账号将获得「\textbf{耀考认证导师}」标识，可自主开启接单状态。

  \item 等待学生预约，系统自动确认
  \item 准时进入 ClassIn 在线教室开始授课

\section{常见问题 FAQ}

\setcounter{subsection}{0}
\renewcommand{\thesubsection}{Q\arabic{subsection}}

\subsection{审核一般需要多长时间？}

正常情况下 \textbf{3 个工作日}内完成审核，高峰期可能略有延迟。提交后可在「个人中心 \arrowright\ 认证状态」实时跟踪进度。

\subsection{被拒绝后还能重新申请吗？}

\textbf{可以，不限次数。}审核拒绝后平台会说明具体原因，补充材料后重新提交，审核结果与上次独立计算。

\subsection{必须要有教师资格证吗？}

\textbf{不是强制要求。}平台综合评估学历背景、专业能力与辅导经历，有教学资质证书有助于通过，但不是唯一标准。

\subsection{可以同时辅导多个科目吗？}

\textbf{可以。}申请时可选择多个课程体系和科目，建议只填写真正擅长的科目，以保证教学质量。

\subsection{学生预约后如何上课？}

平台集成了 \textbf{ClassIn} 在线教室，预约确认后可直接在平台内进入教室上课，需安装 ClassIn 客户端（免费，支持 Windows/Mac/iOS/Android）。

\subsection{平台会抽成吗？收费标准是什么？}

平台按比例收取\textbf{服务费}（具体费率在入驻协议中明确说明），扣除后净收益实时入账，账单透明可查。

\subsection{学生取消预约，我的收益有保障吗？}

\textbf{有保障。}课前约定时间内取消，导师可获全额或部分补偿，具体规则参阅平台服务协议取消条款。

\subsection{海外导师可以申请吗？}

\textbf{可以。}身份证明使用护照，学历证书附官方认证翻译件即可，时区差异可通过灵活排班协调。

\section{重要注意事项}

\subsection{材料真实性}

\begin{warnbox}
提交的所有材料必须\textbf{真实有效}。平台将对关键信息进行核查，包括学历信息院校核实、证件有效性验证等。
\end{warnbox}

\subsection{个人信息保护}

平台严格遵守《个人信息保护法》，对提交的身份证件、学历证书等敏感信息进行\textbf{加密存储}，仅用于资质审核，不向第三方披露。

\subsection{平台行为规范}

成为认证导师后，须遵守以下规范：

\begin{enumerate}
  \item \textbf{准时上课}：提前 5 分钟进入教室，不得无故迟到或缺课
  \item \textbf{教学质量}：保持专业授课态度，不得恶意刷单
  \item \textbf{不得私下联系}：禁止绕过平台与学生进行私下交易
  \item \textbf{内容合规}：授课内容须合法合规，禁止传播违规内容
\end{enumerate}

\begin{errorbox}
违反平台行为规范的导师，将依情节轻重受到警告、限流、暂停或永久封禁处理。
\end{errorbox}
